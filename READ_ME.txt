Auteurs : Pauline Bazelle, Justine Bosc, Eléa Greugny, Maria Prieto-Gonzalez

Il faut lancer le serveur et les clients depuis un terminal.
Pour le serveur : python serveur.py <port>
Pour le client : python client.py <adresse IP serveur> <port> w h nbagents presitant pmort pinfecte

exemple de paramètres : 20 10 100 0.01 0.001 1.0

Chaque simulation doit être lancée depuis une adresse IP différente.
